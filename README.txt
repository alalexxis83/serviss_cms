CMS РЕМОНТ - ЗАЩИТА ПО IP (MD5)
================================

ФАЙЛЫ:
------
config.php    - Конфигурация безопасности и функция проверки IP
403.php       - Страница ошибки "Доступ запрещен"
index.php     - Главная страница (пример использования)
get_hash.php  - Вспомогательный файл для получения MD5 хеша

УСТАНОВКА:
---------
1. Загрузите get_hash.php на сервер
2. Откройте его в браузере
3. Скопируйте полученный MD5 хеш
4. Вставьте хеш в config.php в массив $allowed_ips
5. УДАЛИТЕ get_hash.php!
6. Подключайте config.php во все защищаемые страницы

ИСПОЛЬЗОВАНИЕ:
------------
В начале каждого PHP-файла:

<?php
require_once 'config.php';
checkIpAccess();
// ваш код...
?>

ДОБАВЛЕНИЕ НОВЫХ IP:
------------------
- Временно добавьте в config.php: echo md5($_SERVER['REMOTE_ADDR']); exit;
- Или используйте get_hash.php (потом удалите)

ЛОГИ:
-----
Все попытки несанкционированного доступа записываются в:
logs/security.log
